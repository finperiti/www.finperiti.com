Implementation Details

1. Canonical URLs

Added unique canonical URL to every page to prevent duplicate content penalties and guide search engines to the authoritative version.

2. robots.txt & XML Sitemap

robots.txt — Search engine crawling directives optimizing crawl budget. sitemap.xml — Complete page map with priority levels (home: 1.0, pricing: 0.9, about: 0.8, contact: 0.7, legal: 0.5).

3. Custom 404 Page

Professional error page with proper SEO metadata, navigation, and CTAs. Marked as non-indexable to prevent ranking.

4. Meta Tags

  • Open Graph: og:title, og:description, og:image, og:url, og:type, og:locale
  • Twitter Card: Twitter preview optimization (summary_large_image)
  • Standard: robots directive (index, follow), language, revisit-after

Schema Type Pages Purpose

  • Headquarters: Dublin, Ireland (marked in schema)
  • Service Areas: IE, GB, MT
  • Locale: en_GB for European market
